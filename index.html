HTML elements references

Basic elements

Tag Name Description
<h> Headings h1-6 serves as headins on webpages. the larger the number after the "h" the smaller the size of the heading by default. There are 6 different versions.
<p> Paragraph Paragraph tags are used to insert paragraph length blocks of text into websites. They are "normal" sized and are used to contain most of the text present on websites.
<img> Image tag the image tag is used to include images in websites. They src attribute to find the source of the image, and alt to display text if it cannot be loaded
<li> Organized list tag This tag creates lists where added elements are automatically numbered
<ul> Unorganized list tag This element creates lists where the elements are not automatically numbered.

Media

Tag Name Description
<img> Image tag This tag is used to insert an image. It has two requirements: src and alt.
src source attribute This attribute is used to determine the source of the img in the image tag. It can either link a local file, in which case it uses a ./ prefix or link to a url. It must be enclosed on quotes
alt Alt attribute Display text to either visually impaired users or in cases when the text cannot be displayed
